## How to Install (No App Store Required)

### iPhone / iPad
1. Upload these files to any web host (see hosting options below)
2. Open the URL in **Safari** (must be Safari, not Chrome)
3. Tap the **Share** button ⬆️ at the bottom
4. Scroll down and tap **"Add to Home Screen"**
5. Tap **Add** — Focal appears on your home screen like a native app

### Android
1. Open the URL in **Chrome**
2. Tap the **3-dot menu** in the top right
3. Tap **"Add to Home Screen"** or **"Install App"**
4. Tap **Install**

### Desktop (Mac / Windows / Linux)
1. Open the URL in **Chrome** or **Edge**
2. Look for the **install icon** ⊕ in the address bar
3. Click **"Install Focal"**
4. The app opens in its own window, no browser chrome

## Free Hosting Options for Beta Testing

### Option 1 — GitHub Pages (Recommended, Free)
1. Go to github.com and create a free account
2. Create a new repository named `focal`
3. Upload all files in this folder
4. Go to Settings → Pages → Branch: main → Save
5. Your app is live at: `https://YOUR-USERNAME.github.io/focal`

### Option 2 — Netlify (Free, Drag & Drop)
1. Go to netlify.com
2. Drag this entire folder onto their deploy page
3. Get an instant URL like `focal.netlify.app`

### Option 3 — Vercel (Free)
1. Go to vercel.com
2. Import your GitHub repo
3. Auto-deploys on every update
